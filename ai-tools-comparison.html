📦 工具详细介绍

🔬 OpenAI Codex

有免费额度 需代理访问

Codex 是 OpenAI 推出的 AI 编程工具，基于 GPT-5 模型，专门针对编程任务优化。它可以独立运行在终端中，也支持 VS Code 和 JetBrains IDE 插件。

主要功能：

  • Codex CLI：终端命令行工具，支持自然语言编程
  • IDE 插件：支持 VS Code、JetBrains 系列（IDEA、Pycharm 等）
  • 项目级理解：能理解整个代码库，进行跨文件修改
  • MCP 插件支持：可扩展各种功能
  • 价格：有免费额度，付费约 $20/月
国内使用注意： 需要代理/VPN 才能正常使用，支付需要国外信用卡。

🧠 Claude Code (OpenClaw)

$20-200/月 需代理访问

Claude Code 是 Anthropic 推出的终端 AI 编程代理（2026年初更名为 OpenClaw），被称为目前最强的终端 AI 编程工具。它不是简单的代码补全，而是真正能自主理解代码库、执行复杂任务的 AI 代理。

主要功能：

  • 终端 AI 代理：在命令行中运行，通过自然语言指令完成任务
  • 项目级理解：能读取、分析、修改整个代码库
  • Git 工作流：自动处理 commit、branch 等 Git 操作
  • 多模型选择：支持 Opus 4、Haiku 等模型
  • 价格：
    • Pro：$20/月（200次快速请求/周）
    • Max：$100/月（无限制）/ $200/月（最强模型）
国内使用注意： 需要代理/VPN，支付需要国外信用卡或代购。

💬 ChatGPT

免费/付费 需代理访问

ChatGPT 是 OpenAI 推出的对话式 AI，2022年底发布后掀起生成式 AI 热潮。虽然主要是对话工具，但通过 GPTs 和 Code Interpreter 等功能也能辅助编程。

主要功能：

  • 对话交互：通过自然语言问答获取编程帮助
  • 代码生成：可以直接生成各种语言的代码
  • GPTs：创建定制化的 AI 助手
  • Code Interpreter：可执行代码、进行数据分析
  • o1/o3 模型：强大的推理能力，适合复杂问题
  • 价格：免费版够用，Plus $20/月，Pro $200/月
国内使用注意： 需要代理/VPN，国内手机号无法注册。

⚡ Cursor

免费/付费 部分限制

Cursor 是基于 VS Code 的 AI 原生代码编辑器，深度集成 GPT-4、Claude 等大模型。它不是插件，而是全新的 IDE，AI 深度融入编辑体验。

主要功能：

  • AI 原生 IDE：从底层融入 AI，不是插件
  • 智能补全：基于项目上下文的代码补全
  • Chat 模式：侧边栏对话，询问代码问题
  • Edit 模式：AI 辅助代码编辑
  • Terminal 集成：终端也能智能交互
  • 价格：
    • Hobby：免费，每月 2000 次请求
    • Pro：$20/月，无限请求
    • Business：$40/月/用户
国内使用注意： 2025年7月起对中国大陆断供 Claude、Gemini 等核心模型，付费用户可能受影响。建议关注官方政策或使用国内替代品。

🔗 GitHub Copilot

$10/月 ✅ 国内可用

GitHub Copilot 是微软/GitHub 推出的 AI 代码补全工具，业界老牌选手，与 VS Code、JetBrains 等主流 IDE 无缝集成。

主要功能：

  • 代码补全：基于上下文的智能补全
  • 多语言支持：支持所有主流编程语言
  • IDE 集成：VS Code、JetBrains、Visual Studio 等
  • GitHub Copilot Chat：对话式编程助手
  • 企业版：企业版 $19/月，有安全合规保障
  • 价格：个人版 $10/月，学生/开源免费

🐱 通义灵码

完全免费 ✅ 国内首选

通义灵码是阿里云推出的 AI 编程助手，专门针对中文开发者优化，完全免费！是当前国内最受欢迎的 AI 编程工具之一。

主要功能：

  • 智能补全：代码补全、续写
  • 自然语言生成代码：用中文描述需求生成代码
  • 代码解释：选中代码即可解释
  • 阿里云集成：深度集成阿里云生态
  • IDE 支持：VS Code、JetBrains 全支持
  • 价格：完全免费！

🔍 百度 Comate

免费/付费 ✅ 国内可用

百度 Comate 是百度智能云推出的 AI 编程助手，基于文心大模型，针对中文场景优化。

主要功能：

  • 代码补全：智能代码补全
  • 代码生成：自然语言生成代码
  • 代码优化：代码优化建议
  • 企业版：支持私有化部署
  • 价格：个人免费，企业付费

  • 高性能：代码能力对标 GPT-4

🎯 如何选择？

🎓 学生 / 初学者

推荐：通义灵码、百度 Comate、GitHub Copilot（学生免费）

完全免费，中文体验好，有问题容易找到帮助。

💼 专业开发者

推荐：Claude Code > Codex > Cursor > GitHub Copilot

追求效率和高级功能，愿意为此付费。Claude Code 和 Codex 是目前最强的两个 CLI 工具。

🏢 企业用户

推荐：GitHub Copilot Business/Enterprise、百度 Comate、通义灵码

需要考虑数据安全、合规、私有化部署等企业级需求。

🇨🇳 国内开发者

首选：通义灵码、百度 Comate、DeepSeek Coder

网络稳定、中文优化好、完全免费/价格实惠。如果需要更强功能再考虑代理访问国外工具。

📊 总结

一句话总结：
  • 想省心免费用 → 通义灵码
  • 追求最强 AI 编程能力 → Claude Code / Codex
  • 想要 AI 深度融合的编辑器 → Cursor
  • 老牌稳健选择 → GitHub Copilot
  • 支持国产 → 通义灵码 / 百度 Comate / DeepSeek
⚠️ 特别提醒：
  • 国外工具（Codex、Claude Code、ChatGPT、Cursor）在国内使用需要代理，并可能面临支付障碍和政策风险
  • 2025年7月起 Cursor 对中国大陆限制部分功能，使用前请务必了解最新政策
  • 国内工具虽然免费，但在某些高级功能上可能与国外顶级工具存在差距
